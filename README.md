## 🚀 Overview
Lovart is a dedicated web application focusing on digital aesthetics and interactive SEO-friendly tools. It is designed with a clean architecture and optimized for modern development workflows.

- **Frontend:** HTML5, CSS3, EJS
- **Build System:** Custom Node.js build scripts

## 📂 Project Structure
- `src/`: Source code of the application.
- `scripts/`: Build and automation scripts.
- `public/`: Static assets and processed files.
