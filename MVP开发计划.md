# 国产AI图片生成MVP开发计划

## 项目概述
创建一个简化的AI图片生成网站，使用国产AI模型（豆包/文心一格），采用最简单的技术栈快速验证市场。

## 技术栈
- **前端**: HTML + CSS + JavaScript (原生)
- **后端**: Node.js + Express
- **数据库**: SQLite
- **AI模型**: 豆包(通义千问) / 文心一格
- **部署**: 国内云服务器 (阿里云/腾讯云)

## 页面设计
**主页面布局:**
- 顶部：Logo + 登录/注册 + 用户额度显示
- 中间：图片生成区域（输入框 + 生成按钮）
- 底部：历史记录展示 + 页脚信息

## 开发计划 (4周)

### 第1周：基础框架搭建
- 搭建Express服务器
- 配置SQLite数据库
- 创建基础页面结构
- 实现静态文件服务

### 第2周：用户系统
- 用户注册/登录功能
- JWT token认证
- 用户额度管理
- 密码加密存储

### 第3周：AI集成
- 集成豆包/文心一格API
- 图片生成接口
- 图片存储服务
- 异步任务处理

### 第4周：前端完善
- 图片生成界面
- 历史记录展示
- 图片下载功能
- 错误处理优化

## 核心功能流程

### 用户认证流程
注册 → 邮箱验证 → 登录 → 获取token → 访问受保护资源

### 图片生成流程
用户输入 → 调用AI API → 接收图片 → 保存到本地/云存储 → 更新用户额度 → 返回结果

### 额度管理
- 新用户注册：10张免费额度
- 生成图片：扣除1张额度
- 额度用完：提示购买
